## How to Build 💻

### Requirements 🔧
- [**MAKE**](https://cmake.org/install/)
- [**GCC**](https://gcc.gnu.org/install/)

- First, clone repository
```sh
git clone https://github.com/mateo-astudillo/tp-c.git
```
- Second, move to directory
```sh
cd tp-c
```
- Third, run the following command
```sh
make
```

## How to run 🚀
#### Windows 🪟

- Run the .exe file
```powershell
.\bin\tp.exe
```
or (in bin directory)
```
.\tp.exe
```

#### Linux 🐧
- Run the bin file
```sh
./bin/tp
```
or (in bin directory)
```
./tp
```
